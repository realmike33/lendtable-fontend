## Getting Started

First create .env.local file using env var from .env.example file

Install dependents

```bash
npm insall
# or
yarn
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
